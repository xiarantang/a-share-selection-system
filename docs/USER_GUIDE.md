# 🏦 A 股智能选股系统 — 小白使用指南

> 不会写命令？没关系，双击就能用。本指南假设你只会用鼠标。

## 一、这是什么

一个帮你从 A 股 55 只精选股票里挑出「值得关注」候选的程序。不推荐买卖，只帮你做初步筛选。

选完后会告诉你：排名、评分、风险高低、数据可不可靠。

**不构成投资建议。** 仅供研究学习。

---

## 二、第一次使用（只需做一次）

### 安装备用数据通道（约 1 分钟）

双击项目里的这个文件：

```
scripts/install_fallback.command
```

屏幕上会显示安装进度。这一步是下载 A 股历史数据脚本（约 120 条 K 线），确保系统即使在网络不稳定时也能正常出结果。

> 💡 只需安装一次，以后使用不用再装。

---

## 三、启动系统

双击：

```
start_ui.command
```

等几十秒，浏览器会自动打开一个页面。

如果浏览器没自动打开，手动在浏览器地址栏输入：

```
http://localhost:8501
```

看到标题「🏦 A股智能选股系统」就说明启动成功了。

> ✅ 双击启动路径已通过冒烟验收（P8.6-3），兼容 macOS 环境。UI 已清理废弃 API 警告（`use_container_width` / `applymap`），截图已更新（P8.6-1/2）。

### 如果页面显示「需要先安装备用数据通道」

说明第二步的安装没有完成。请关闭页面，双击 `scripts/install_fallback.command`，安装完成后再重新双击 `start_ui.command`。

---

## 四、开始选股

页面左边有几个选项，保持默认就好：

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 股票池 | static (55只精选) | 不用改 |
| 选股策略 | 默认规则策略 | 当前为规则策略选择，不改变评分公式 |
| 扫描数量 | 10 | 扫 10 只，改大会更慢 |
| 展示数量 | 5 | 结果显示前 5 名 |
| 数据起始日期 | 2024-01-01 | 不用改 |

然后点击 **「🚀 开始选股」** 按钮。

**等 30-60 秒**，页面会显示进度。请不要关闭页面。

---

## 五、看结果

选股完成后，页面主区域会直接显示：

结果区会显示当前使用的策略名称、适用场景和风险提醒，帮助你了解本次选股使用的规则。

### 1. 数据覆盖情况

- 一行字告诉你数据的实际区间（比如 `2025-11-13 ~ 2026-05-15`）
- 如果看到「覆盖不全」提示，**不是系统坏了**，只是可用的历史数据较短（约 120 条 K 线）。系统会自动降低评分和置信度，结果仍可参考。

### 2. 验证摘要（彩色卡片）

包含：整体质量（良好 / 需谨慎 / 差）、平均评分、覆盖不足比例、高风险数量。

### 3. 点击「🎯 候选表格」Tab

能看到排名列表：代码、名称、评分、决策（强观察 / 观察 / 中性 / 回避）、风险等级、置信度、数据区间。

每行底部有逐只详情，点击展开可以看到因子得分和具体原因。

### 4. 点击「📋 验证摘要」Tab

更详细的验证数据：置信度分布、决策分布、行业分布。

### 5. 点击「📈 历史复盘」Tab

需要对选股结果做历史回看时，切换到该 Tab 后点击左侧「📈 历史复盘」按钮。会告诉你过去类似评分的股票实际表现如何。

> ⚠️ 历史复盘是对已有数据的 In-Sample 验证，不代表未来收益。

### 6. 点击「📄 报告」Tab

生成一份 Markdown 报告，可直接阅读或复制保存。

## 六、常见问题

**Q: 点了「开始选股」一直没反应？**
A: 正常需要等 30-60 秒。如果超过 2 分钟还没反应，刷新页面（F5）再试一次。

**Q: 选股结果显示的都是「覆盖不全」？**
A: 正常。当前备用数据通道约 120 条 K 线，系统已自动降低评分。这不代表结果不能用，但建议多观察几次。

**Q: akshare 失败是不是系统坏了？**
A: 不是。akshare 受网络影响经常失败，系统会自动切换到备用数据通道。只要能看到候选表格，就是正常运行。

**Q: 怎么更新数据？**
A: 在左侧「数据起始日期」改日期即可。但当前备用数据通道只有约 120 条 K 线，改更早的日期会导致更多「覆盖不全」标记。

**Q: 可以加更多股票吗？**
A: 默认 static 股票池包含 55 只精选股票。在左侧「扫描数量」调大即可扫更多只。

>
> 当前选股模型为规则因子评分（多因子分组，满分 100），不是收益预测。A 股市场风险较高，投资需谨慎。
>
> 数据来源：akshare + 多源 fallback（腾讯/新浪/雪球/东财）+ 本地缓存。当 akshare 不可用时会自动降级到备用数据源。若出现「覆盖不全」提示，表示数据起始日晚于请求起始日，模型已自动下调评分和置信度。数据覆盖不足时不构成选股建议。
