# 发布后问题记录

> 创建时间：2026-05-19
> 阶段：P10.3（发布后观察期）
> 基线：v0.6-rc1

---

## 1. 文档目的

记录 v0.6-rc1 发布后观察期内的用户反馈、问题发现、复现步骤和处理状态。便于后续判断是否需要 v0.6-rc2 或可以直接升为 v0.6。

## 2. 使用原则

- 只记录问题和复盘，不改变评分/排序/数据链路
- 不记录投资决策相关内容（本系统仅供研究学习，不构成投资建议）
- 每条记录一个独立问题，便于追踪和分级
- 处理完成后及时更新状态

## 3. 问题分级

| 分级 | 含义 | 示例 |
|------|------|------|
| 阻断 | 影响核心功能正常使用 | 双击启动失败、选股无法完成 |
| 重要 | 影响体验但不阻断核心流程 | 截图与实际 UI 不一致、提示文案不清晰 |
| 一般 | 小体验问题、建议改进 | 排版间距、字体大小 |
| 观察 | 暂不需要处理，持续关注 | 偶发网络超时后自动恢复 |

## 4. 状态定义

| 状态 | 含义 |
|------|------|
| 待复现 | 已收到反馈，尚未确认是否可复现 |
| 已确认 | 已复现或已确认问题存在 |
| 处理中 | 正在修复或改进 |
| 已修复 | 已在某个阶段修复并验收 |
| 暂不处理 | 评估后决定当前不修复，记录原因 |
| 已关闭 | 已修复并验证，或经评估不需要修复 |
