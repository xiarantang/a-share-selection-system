# UI 验收路径（小白点击流）

> 双击 `start_ui.command` → 选参数 → 开始选股 → 看结果。

## 免责声明

本系统仅供研究学习，不构成投资建议。

## 前置条件

- macOS 环境
- 已安装数据 fallback（`~/.agents/skills/a-share-data/` 目录存在）
- 首次运行会自动创建 `.venv` 并安装轻量依赖

## 小白路径

1. 双击项目根目录下的 `start_ui.command`
2. 浏览器自动打开 → 在左侧选参数（保持默认即可）
3. 点击「🚀 开始选股」
4. 等待结果 → 看 Top3 速览 / 候选表格 / 逐只详情

---

## 验收步骤

### 1. 启动系统

双击项目根目录下的 **`start_ui.command`**，终端会显示：

```
========================================
  🏦 A 股智能选股系统
  仅供研究学习，不构成投资建议
========================================
```

自动检测 Python3 → 创建/确认虚拟环境 → 安装依赖 → 启动 Streamlit。

浏览器自动打开或手动访问 **http://localhost:8501**。

### 2. 首页引导

打开后首页应显示：

- 标题「🏦 A股智能选股系统」
- 副标题「选参数 → 开始选股 → 看结果 · 小白也能用 · 仅供研究学习，不构成投资建议」
- 引导卡片：① 在左侧选参数、② 等待结果
- 系统就绪横幅：「✅ 系统就绪 — 点击左侧开始选股」

左侧栏参数面板可见，包含：股票池、策略选择器（默认规则策略）、扫描数量、展示数量、起始日期。

### 3. 选股

1. 左侧栏「选择股票池」：保持默认 **static (55只精选)**
2. 左侧栏「选股策略」：保持默认 **默认规则策略**
3. 扫描数量保持默认 10，展示数量保持默认 10
4. 数据起始日期保持默认 2024-01-01
5. 点击 **🚀 开始选股** 按钮

**预期**：
- 按钮点击后显示 "⏳ 正在选股中" 加载动画
- 完成后弹出绿色成功提示（如 "✅ 选股完成！成功 10/10 只"）
- 如果 akshare 不可用，提示 "数据来自备用源（akshare 暂不可用），覆盖可能不全"（非崩溃）

### 4. 查看 Top3 速览

选股完成后，结果页上方应显示 **Top3 速览卡片**：

- 三张卡片分别显示 🥇🥈🥉 排名
- 每张卡片包含：代码名称、评分、中文决策标签（强观察/观察/中性/回避）、风险等级（带颜色）
- 每张卡片包含一句话解释（explain.summary）
- 数据概览区域：股票池/数据区间/K线数据源/整体质量

### 5. 查看候选表格

自动切换到 **🎯 候选表格** Tab：

- 显示 Top N 候选的排行表格
- 表格包含：排名、代码、名称、行业、评分、决策（中文）、风险（中文）、置信度（中文）、数据区间
- 决策列有颜色区分（绿色=强观察/浅绿=观察/红色=回避）
- 风险列显示 🟢低风险 / 🟡中风险 / 🔴高风险
- 置信度列显示 高置信度 / 中置信度 / 低置信度
- **数据区间** 列显示每只股票的实际数据起止日期（含"约近 X 个月"提示）
- **覆盖** 列显示 ✓（正常）或 ⚠️不全（覆盖不足）
- 策略/适用场景/风险提醒在数据概览下方可见

### 6. 查看逐只详情

在候选表格下方，展开任意股票的详情：

- 标题显示中文决策和风险（如「观察 · 低风险」）
- 显示一句话解释（explain.summary）
- 显示评分、决策、风险、置信度（均为中文标签）
- 显示主要加分项、主要风险项、可靠性说明
- 因子得分带中文图标（📈趋势/🚀动量/📊量能/🛡️风控/📋数据质量/🔮形态）
- 技术指标详情在折叠区（默认收起）
- 显示数据区间（actual_start ~ actual_end，含"约近 X 个月"提示）
- 如果覆盖不全，显示 ⚠️ 警告及白话解释

### 7. 查看验证摘要

切换到 **📋 验证摘要** Tab：

- 显示整体质量（🟢 good / 🟡 usable_with_caution / 🔴 poor）
- 整体质量下方有中文解释
- 显示总候选数、选股成功、覆盖不足率、平均分/最高分
- 显示置信度分布、决策分布、风险分布
- 显示行业分布柱状图

### 8. 运行历史复盘

1. 点击左侧栏 **📈 历史复盘** 按钮（需要先有选股结果，否则按钮灰色）
2. 等待加载（较慢，需要重新拉取数据）

**预期**：
- 加载完成后弹出成功提示
- 自动切换到 **📈 历史复盘** Tab
- 显示复盘汇总卡片（验证总数/跳过/Win/Flat/Loss + 平均收益/回撤/最佳/最差）
- 下方显示逐只复盘详情表格

### 9. 查看报告

切换到 **📄 报告** Tab：

- 自动生成完整 Markdown 日报
- 包含市场概况、验证摘要、批量选股结果、历史窗口复盘、免责声明
- 底部提示报告已保存路径

### 10. 确认数据覆盖提示

在整个流程中，应能直观看到：
- 候选表格的「数据区间」列（含"约近 X 个月"提示）
- 数据概览区域的完整数据区间
- 逐只详情中的覆盖不足警告
- 验证摘要中的覆盖不足率

**不需要翻日志就能了解数据覆盖情况。**
