# 人工 UI 验收清单

> 给小白/人工验收用。双击 `start_ui.command` → 选参数 → 开始选股 → 看结果。

## 免责声明

本系统仅供研究学习，不构成投资建议。

## 前置条件

- macOS 环境
- `.venv/bin/streamlit` 可用
- requirements-ui 已包含 baostock，skill_fallback 为可选第三级兜底

## 小白路径

1. 双击 `start_ui.command`
2. 浏览器自动打开 → 在左侧选参数（保持默认即可）
3. 点击「🚀 开始选股」
4. 等待结果 → 看 Top3 速览 / 候选表格 / 逐只详情

---

## 验收步骤

### 1. 启动

- [ ] 双击 `start_ui.command`
- [ ] 终端打印 "🏦 A股智能选股系统" 横幅
- [ ] 自动检测 Python3 ✅ / 虚拟环境 ✅ / 依赖 ✅
- [ ] 浏览器打开 `http://localhost:8501`

### 2. 首页

- [ ] 页面标题「🏦 A股智能选股系统」
- [ ] 标题下方副标题：「选参数 → 开始选股 → 看结果」
- [ ] 首页引导卡片可见：① 在左侧选参数、② 等待结果
- [ ] 系统就绪状态横幅：「✅ 系统就绪 — 点击左侧开始选股」
- [ ] 左侧栏参数面板可见（股票池、策略、扫描数量、展示数量、起始日期）
- [ ] 底部有免责声明

### 3. 选股

- [ ] 左侧栏股票池默认 `static`（55只精选）
- [ ] 左侧栏策略选择器显示「默认规则策略」
- [ ] 点击「🚀 开始选股」按钮
- [ ] 出现「⏳ 正在选股中」加载动画
- [ ] 等待约 10-60秒（取决于网络）
- [ ] **出现绿色提示「✅ 选股完成！成功 X/X 只」** ← 关键通过点

### 4. Top3 速览

- [ ] 结果页上方出现 Top3 速览卡片（🥇🥈🥉）
- [ ] 每张卡片显示：代码名称、评分、中文决策标签、风险等级（带颜色）
- [ ] 每张卡片包含一句话解释（explain.summary）
- [ ] 数据概览区域显示：股票池、数据区间、K线数据源、整体质量

### 5. 候选表格

- [ ] 表格列出 Top N 候选股票
- [ ] 列包含：排名、代码、名称、行业、评分、决策（中文）、风险（中文）、置信度（中文）、数据区间
- [ ] 决策列有颜色区分（绿色=强观察/浅绿=观察/红色=回避）
- [ ] 风险列显示：🟢低风险 / 🟡中风险 / 🔴高风险
- [ ] 置信度列显示：高置信度 / 中置信度 / 低置信度
- [ ] 「覆盖」列显示 ✓（正常）或 ⚠️不全
- [ ] 策略/适用场景/风险提醒在数据概览下方可见

### 6. 逐只详情

- [ ] 点击某行下方的展开箭头
- [ ] 标题显示中文决策和风险（如「观察 · 低风险」）
- [ ] 显示评分、决策、风险、置信度（均为中文标签）
- [ ] 显示一句话解释（explain.summary）
- [ ] 显示主要加分项、主要风险项、可靠性说明
- [ ] 因子得分带中文图标（📈趋势/🚀动量/📊量能/🛡️风控/📋数据质量/🔮形态）
- [ ] 技术指标详情在折叠区（默认收起）
- [ ] 显示数据源、K线条数、数据区间
- [ ] 如果覆盖不全，显示 ⚠️ 白话警告

### 7. 验证摘要

- [ ] 点击「📋 验证摘要」Tab
- [ ] 显示整体质量（🟢 good / 🟡 usable_with_caution / 🔴 poor）
- [ ] 质量下方有中文解释
- [ ] 显示总候选数、选股成功、覆盖不足率、平均分/最高分
- [ ] 显示置信度分布、决策分布、风险分布
- [ ] 显示行业分布柱状图

### 8. 历史复盘

- [ ] 左侧栏「📈 历史复盘」按钮可点击（选股后启用）
- [ ] 点击后弹出加载动画
- [ ] 完成后显示复盘汇总（Win/Flat/Loss 计数 + 平均收益/回撤）
- [ ] 逐只复盘详情表格可见

### 9. 报告

- [ ] 点击「📄 报告」Tab
- [ ] 完整 Markdown 日报渲染
- [ ] 包含选股结果、验证摘要、复盘结果、免责声明

### 10. 数据覆盖可视化（全程）

- [ ] 候选表格「数据区间」列每只可见（含"约近 X 个月"提示）
- [ ] 数据概览区域显示整体数据范围
- [ ] 逐只详情中可看每条实际数据起止日期
- [ ] 如果覆盖不全，各层级均有警告提示
- [ ] **不翻日志就能了解数据覆盖** ← 最终验收目标
