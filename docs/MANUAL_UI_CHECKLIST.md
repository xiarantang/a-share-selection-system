# 人工 UI 验收清单

> 给小白/人工验收用。双击 `start_ui.command` 后按此清单逐项检查。

## 前置条件

- macOS 环境
- `.venv/bin/streamlit` 可用
- 已安装数据 fallback（`~/.agents/skills/a-share-data/` 目录存在）

## 验收步骤

### 1. 启动

- [ ] 双击 `start_ui.command`
- [ ] 终端打印 "🏦 A股智能选股系统" 横幅
- [ ] 自动检测 Python3 ✅ / 虚拟环境 ✅ / 依赖 ✅
- [ ] 浏览器打开 `http://localhost:8501`

### 2. 首页

- [ ] 页面标题「A股智能选股系统」
- [ ] 三列引导卡片可见：①选参数、②开始选股、③看结果
- [ ] 左侧栏自动展开，参数面板可见
- [ ] 底部有免责声明

### 3. 选股

- [ ] 左侧栏股票池默认 `static`（55只精选）
- [ ] 点击「🚀 开始选股」按钮
- [ ] 出现「⏳ 正在选股中」加载动画
- [ ] 等待约 10-60秒（取决于网络）
- [ ] **出现绿色提示「✅ 选股完成！成功 X/X 只」** ← 关键通过点

### 4. 候选表格

- [ ] 表格列出 Top N 候选股票
- [ ] 列包含：排名、代码、名称、行业、评分、决策、风险、置信度、数据区间
- [ ] 「覆盖」列显示 ✓（正常）或 ⚠️不全
- [ ] 表格上方有「📊 数据区间: ... | K线: ... | 覆盖不全: ...」信息条
- [ ] 表格上方有覆盖摘要卡片（数据范围/最少K线/最多K线/平均K线/覆盖不全数）

### 5. 逐只详情

- [ ] 点击某行下方的展开箭头
- [ ] 显示评分、决策、风险、置信度
- [ ] 显示数据源、K线条数、数据区间
- [ ] 六因子得分和因子数值可见
- [ ] 如果覆盖不全，显示 ⚠️ 白话警告

### 6. 验证摘要

- [ ] 点击「📋 验证摘要」Tab
- [ ] 显示整体质量（🟢 good / 🟡 usable_with_caution / 🔴 poor）
- [ ] 质量下方有中文解释
- [ ] 显示置信度分布、决策分布、风险分布
- [ ] 显示行业分布柱状图

### 7. 历史复盘

- [ ] 左侧栏「📈 历史复盘」按钮可点击（选股后启用）
- [ ] 点击后弹出加载动画
- [ ] 完成后显示复盘汇总（Win/Flat/Loss 计数 + 平均收益/回撤）
- [ ] 逐只复盘详情表格可见

### 8. 报告

- [ ] 点击「📄 报告」Tab
- [ ] 完整 Markdown 日报渲染
- [ ] 包含选股结果、验证摘要、复盘结果、免责声明

### 9. 数据覆盖可视化（全程）

- [ ] 候选表格「数据区间」列每只可见
- [ ] 表格上方信息条显示整体数据范围
- [ ] 覆盖摘要卡片显示 K 线统计
- [ ] 逐只详情中可看每条实际数据起止日期
- [ ] **不翻日志就能了解数据覆盖** ← 最终验收目标

---

## 验收通过标准

| 检查项 | 门槛 |
|--------|------|
| 启动成功 | 浏览器能打开 8501 |
| 选股成功 | 绿色完成提示出现 |
| 候选表格 | 含数据区间列 |
| 覆盖信息条 | 📊 开头的 data range信息可见 |
| 全覆盖卡片 | K线统计可见 |
| 验证摘要 | 质量标签有中文解释 |
| 复盘 | 能正常跑出结果 |
| 报告 | Markdown 完整渲染 |
| 无崩溃 | 整个流程无异常报错 |

全部打 ✅ 即为通过。
